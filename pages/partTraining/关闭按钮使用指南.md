# 关闭按钮使用指南

## 方案：直接使用 SVG 静态资源

推荐使用项目提供的通用关闭按钮 SVG 图标，无需封装组件，无需额外容器。

---

## 静态资源路径

```
/static/icons/general/btn_general_close.svg
```

---

## 使用方法

### 1. 模板代码

```vue
<template>
  <view class="your-container">
    <!-- 关闭按钮：直接使用 image 标签 -->
    <image 
      class="close-btn" 
      src="/static/icons/general/btn_general_close.svg" 
      mode="aspectFit"
      @click="handleClose"
    />
  </view>
</template>
```

### 2. 样式代码

```scss
/* 关闭按钮样式 */
.close-btn {
  position: absolute;  /* 绝对定位 */
  top: 8rpx;           /* 距离顶部 */
  right: 8rpx;         /* 距离右侧（或改为 left: 8rpx 放左边） */
  z-index: 10;         /* 层级，确保在最上层 */
  width: 36rpx;        /* 按钮宽度 */
  height: 36rpx;       /* 按钮高度 */
  cursor: pointer;     /* 鼠标指针 */
  user-select: none;   /* 禁止选中 */
}

/* 点击效果：透明度变化 */
.close-btn:active {
  opacity: 0.7;
}
```

### 3. 脚本代码

```javascript
// 关闭事件处理
const handleClose = () => {
  console.log('关闭按钮被点击')
  // 你的关闭逻辑...
  uni.showToast({
    title: '已关闭',
    icon: 'none'
  })
}
```

## 样式调整参考

### 位置调整
```scss
/* 左上角 */
top: 8rpx;
left: 8rpx;

/* 右上角 */
top: 8rpx;
right: 8rpx;

/* 右下角 */
bottom: 8rpx;
right: 8rpx;

/* 左下角 */
bottom: 8rpx;
left: 8rpx;
```

### 大小调整
```scss
/* 小尺寸 */
width: 28rpx;
height: 28rpx;

/* 默认尺寸 */
width: 36rpx;
height: 36rpx;

/* 大尺寸 */
width: 48rpx;
height: 48rpx;
```

### 透明度调整
```scss
/* 半透明按钮 */
.close-btn {
  opacity: 0.8;
}

/* 点击时完全不透明 */
.close-btn:active {
  opacity: 1;
}
```

## 优势

✅ **简洁** - 无需封装组件，直接使用  
✅ **干净** - 无额外容器，无背景色  
✅ **灵活** - 样式完全可控  
✅ **轻量** - 使用 SVG 矢量图，清晰且体积小  
✅ **统一** - 全项目使用同一个 SVG 资源  

---

## 注意事项

1. **父容器定位**  
   关闭按钮使用 `position: absolute`，父容器需要设置 `position: relative`

2. **z-index 层级**  
   确保 `z-index` 足够高，避免被其他元素遮挡

3. **点击区域**  
   如果按钮太小难以点击，可以增加 `width` 和 `height`

4. **交互反馈**  
   建议保留 `:active` 伪类的透明度变化，提供点击反馈

---

## 何时不使用此方案

如果需要以下功能，考虑封装成组件：

- 需要动画效果（旋转、缩放等）
- 需要复杂的状态管理
- 需要自定义图标（非 SVG）
- 需要在多个页面复用且样式完全一致

但对于大多数场景，**直接使用 SVG 静态资源是最简单高效的方案**。
